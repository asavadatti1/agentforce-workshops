Workshop Overview

Goal The Starbucks onsite Agentforce workshop took place on June 4th, 2026. The workshop was a hands-on customer engagement designed to help Starbucks take steps toward becoming an "Agentic Enterprise."

The goal was to give the Starbucks team practical experience building agents using the new Agentforce Builder, Agent Script, and Agentforce Vibes.

The workshop was a major success, with Starbucks attendees responding enthusiastically. Key feedback from the customer included:

  • This was exactly what we were looking for
  • Let's dig into Headless
  • This is exciting

Use Case

Pre-Visit Summary for Coffeehouse Consultants (CHCs) Goal of the workshop was to help the cusotmer build an agent that could automatically generate store visit summaries for Coffeehouse Consultants (who manage licensed stores - stores in grocery, airport etc) that contains summary information about the store and possible items to check during the visit including open compliance records, previous visit issues, store performance etc.

Attendees were essentially building a real-world, practical agent focused on the Licensed Store business — which is also the primary area for their upcoming POC!
1
Create a Custom Agent using Agentforce Builder
Create an agent called Nexus agent and add relevant instructions and sub-agents to it.
2
Test the Custom Agent
Test the Nexus agent with relevant questions to ensure it functions as expected.
3
Create a Custom Component using Agentforce Vibes
Create a custom component using Agentforce Vibes to monitor compliance for the licensed stores.
